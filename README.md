
# simple-restful-node

This is a standard implementation of a RESTful API, based on Node.js with Express.js.

### Installation

1. Clone this repo.
   using either <i>Download</i> and extract or `git clone`
2. Install the dependencies
   using `npm install`
3. Start it
   using `npm run start`
